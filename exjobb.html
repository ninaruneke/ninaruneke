Examensarbete

Mitt examensarbete gjordes i samarbete med Tieto, på deras kontor i Umeå. Min uppgift var att undersöka vilka metoder som finns för att göra användartester så man inte ha möjlighet att vara på samma plats som sina användare.

Abstract

Remote usability testing is tests that are performed either moderated or unmoderated with the test facilitator and the test participant at different geographic locations. This master thesis examines different methods for remote usability testing and evaluates which method is most suited for the VAS project at Tieto (VAS is an administration system for health care currently being modernised by Tieto). Based on a preliminary study methods for both moderated and unmoderated remote usability testing were chosen and were later on evaluated by performing tests on a prototype with end-users.

Due to bad timing towards the rest of the VAS project Tieto's test reference group could not participate in any test. Instead developers from Tieto working with the VAS project participated in the moderated tests and end-users without any connection to the project participated in the unmoderated tests. Because of the problems with finding users and some technical problems only four tests were conducted, therefore it is hard to draw any conclusions from the test results. However, the positive response from both test participants and test facilitator towards remote usability testing makes it something to investigate further.
